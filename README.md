# Spring Boot Core Modules 

This repository contains all the **Spring Boot practice projects** I’ve built while learning and exploring different modules of the Spring Boot ecosystem.  
It serves as a collection of hands-on implementations covering various Spring Boot features, configurations, and integrations.

---

##  About
The goal of this repo is to strengthen my understanding of **Spring Boot fundamentals** and provide reusable code snippets for future projects.  
Each folder in this repository represents a different module or mini-project that demonstrates a core concept of Spring Boot.

---

##  Technologies & Tools
- **Java 21** (Core language)
- **Spring Boot** (Core framework)
- **Spring MVC / REST APIs**
- **Spring Data JPA & Hibernate**
- **Spring Security** (basic setups in some modules)
- **Databases**: Oracle 21c, MySQL
- **Maven / Gradle** (depending on the project)
- **Eclipse IDE**
